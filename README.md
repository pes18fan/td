# td

Very simple command-line todo app written in less than 200 sloc of C++. Inspired 
by [this](https://github.com/sioodmy/todo).

## How to use?

Check `td help`.

## Building

Use `make`. If you don't have it, just compile with any C++ compiler; for instance
with `g++`:

```bash
g++ main.cpp -o todo
```
